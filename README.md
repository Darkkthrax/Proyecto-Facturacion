# Proyecto de Facturación
## Descripción
Proyecto universitario sobre un sistema de facturación para un supermercado que interactúa con el inventario del negocio, así como maneja una interfaz gráfica utilizando **Tkinter**.
Genera facturas en formato PDF automáticamente, permite el registro de administradores, empleados y clientes.

Diseñado para sistemas operativos Windows.

**⚠️Aún se encuentra en desarrollo⚠️**

## Tecnologías utilizadas
* Python 3.12

### Librerías
* Tkinter _(Para interfaz gráfica)_
* SQLite3 _(Para manejo de bases de datos SQL)_
* FPDF2 _(Para creación de PDF)_
* Os
* Sys
* bcrypt
* python-dotenv
